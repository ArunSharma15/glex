Detailed Description

A reasonable attempt at implementing a camera interface. Purpose is to change camera position using uniform variables and acceptance of keyboard input. An attempt based on the tutorial www.opengl-tutorial.org/beginners-tutorials/tutorial-6-keyboard-and-mouse/
